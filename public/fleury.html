Fleury's Algorithm

Fleury's Algorithm finds an Euler path by removing a "bridge" or "edge" after traversal. It will continue until all edges are removed.

  • Graph must have 0 or 2 odd vertices
  • 0 odd vertices -> start anywhere
  • 2 odd vertices -> start at one of them
  • Follow edges one at a time
  • If you encounter a bridge (can't go back to the previous vertices) choose the alternative
  • Stop when you run out of edges

    • Complexity: O(n^2) -> Quadratic Time
